# Visualisasi Data & Infografis: Analisis Dataset (Kelompok X / Kelas Y)

Repositori ini merupakan pusat penyimpanan dan dokumentasi untuk pemenuhan tugas Post Test & Prajilid Praktikum Algoritma Pemrograman 2026. Proyek ini mencakup pengolahan data menggunakan Python (Pandas, Matplotlib, Seaborn) serta penerjemahan hasil analisis menjadi infografis edukatif.

## 📂 Komponen Pengumpulan Tugas
Sesuai dengan ketentuan penugasan, repositori ini memuat:
1. **Source Code:** Berkas `.ipynb` yang berisi pengolahan data dan logika visualisasi.
2. **Dataset:** Berkas `.csv` yang digunakan sebagai sumber data analisis.
3. **Desain Infografis:** Berkas final (resolusi tinggi) untuk publikasi media sosial.
